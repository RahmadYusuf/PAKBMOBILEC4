# pamobilekbC4

Penjelasan Aplikasi

Aplikasi Deteksi Hewan (Anjing atau Kucing) adalah aplikasi mobile berbasis Flutter yang dapat mengidentifikasi apakah sebuah gambar termasuk anjing atau kucing menggunakan model Machine CNN.
Aplikasi bertujuan membantu pengguna melakukan klasifikasi gambar secara cepat dan sederhana.

Pengguna cukup mengambil gambar menggunakan kamera atau memilih gambar dari galeri, lalu aplikasi akan memproses gambar tersebut dan menampilkan hasil prediksi.

üéØ Fitur Utama

1.  UI sederhana dan mudah digunakan
2.  Upload gambar melalui kamera
3.  Upload gambar dari galeri
4.  Deteksi hewan menggunakan model ML CNN
5.  Tombol kirim untuk memproses gambar
6.  Menampilkan hasil prediksi secara langsung
